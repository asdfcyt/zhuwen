# 筑文 (ZhuWen)

> 告别"单点"依赖，用系统化思维重构AI写作全流程

## 简介

**筑文**是一款专注于解决写作痛点的系统化AI写作工具。它不是简单的"AI文本生成器"，而是构建了从素材积累、创作构思、内容生成到质量校对、格式导出的完整工作流。

### 核心理念

把写作从"零散用AI"改造为"全流程协同"的系统工程，让每一篇文字都立足于更扎实的资料、更严谨的逻辑、更可靠的流程。

---

## 主要特性

### 🔐 信息脱敏
- 一键添加敏感词列表（人名、地名、机构名、数据）
- 自动加密后发送AI，生成后自动解密
- 让组织放心使用AI写作工具

### 📚 素材库与知识库
- **素材库**：沉淀优秀表达、句式、段落为可复用资产
- **知识库**：上传文章、法规、案例自动向量化，AI在知识边界内工作
- **智能补全**：词典功能 + 自定义库，让AI"越用越懂你"

### 📝 便签系统
- 全局快捷键跨应用快速记录
- 支持Markdown、图片、OCR识别
- 可同步至思源笔记
- 构建"发现→记录→沉淀→调用"完整闭环

### 🔍 联网搜索
- 预设政策法规和理论文章常用网站
- 快捷搜索语法 + 可切换搜索引擎
- 一键阅读模式（去广告）
- 好段落/整篇文章直接保存

### ✍️ 快速写作
1. **生成大纲**：基于文体、风格、字数要求
2. **组装内容**：从素材库、便签拖拽模块
3. **生成文章**：AI围绕内容骨架生成完整文稿

### 🤖 AI助手（双模式）
- **编辑模式**：精细修改已成文段落，调用知识库补充事实数据
- **灵感模式（Deep Plan）**：反向发问澄清需求，生成完整写作计划

### ✅ 智能校对
- **符号格式校对**：中英文符号、格式错误
- **语法表述校对**：句式通顺、逻辑连贯
- **数字逻辑校对**：自动提取数字，校验比例、合计、同比环比

### 📤 智能导出排版
- 公文格式导出（符合排版规范）
- 微信公众号格式导出
- 按模板导出（预设Word模板自动填充）

### 🍅 智能番茄钟
- AI分析行为自动调整时长
- 心流时延长保护专注
- 零散浏览时加快凝聚注意力

## 系统化工作流程

```
前提安全（信息脱敏）
    ↓
积累基础（素材库、便签、知识库）
    ↓
过程优化（快速写作、智能补全、AI伴写与AI助手）
    ↓
质量保障（智能校对）
    ↓
成果落地（智能导出排版）
```
