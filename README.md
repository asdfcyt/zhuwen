# 筑文 (ZhuWen Writer)

**筑文**是一款专为中文用户打造的AI智能写作助手，提供Notion风格的所见即所得编辑体验，集成多种AI模型，支持知识库管理、素材库、文档脱敏等专业写作功能。

## ✨ 核心特性

- 🤖 **多模型AI辅助写作** - 支持OpenAI GPT、Google Gemini、Ollama本地模型
- 📝 **所见即所得编辑器** - 基于Tiptap，支持富文本编辑、表格、图片等
- 🗃️ **智能知识库** - 基于LanceDB向量数据库，支持文档上传和语义搜索
- 📚 **素材库管理** - 写作素材模板系统，支持基因分析和智能应用
- 🏷️ **便签系统** - 写作灵感管理和智能应用
- 🔒 **文档脱敏** - 敏感信息自动识别和替换保护
- 📊 **专注度分析** - 实时分析写作专注度和效率
- 🖥️ **桌面应用** - 基于Electron，支持macOS和Windows

## 🚀 快速开始

### 下载安装

从 [GitHub Releases](https://github.com/YOUR_USERNAME/zhuwen-writer/releases) 下载最新版本：
- **macOS**: 下载 `.dmg` 文件
- **Windows**: 下载 `.exe` 安装包

### 系统要求

- **操作系统**: macOS 10.15+, Windows 10+
- **内存**: 至少4GB RAM（推荐8GB+）
- **存储**: 至少1GB可用磁盘空间
