# 快速开始指南

本指南将帮助您快速上手使用筑文。

## 安装

### macOS

1. 下载 `.dmg` 文件
2. 双击打开，将筑文拖入"应用程序"文件夹
3. 首次运行可能需要在"系统偏好设置 > 安全性与隐私"中允许打开

### Windows

1. 下载 `.exe` 安装包
2. 双击运行安装程序
3. 按照向导完成安装

## 首次使用

### 配置AI模型

1. 点击右上角设置图标
2. 选择"AI设置"
3. 选择您的AI服务提供商（OpenAI、Google Gemini或本地Ollama）
4. 输入API密钥并保存

### 创建文档

1. 点击左侧边栏的"+"按钮
2. 输入文档标题
3. 开始在编辑器中写作

### 使用AI辅助

- **续写**: 输入`++`或按`Tab`键触发AI续写
- **选中改写**: 选中文字后，使用右键菜单中的改写选项
- **AI助手**: 按`Cmd+J`(macOS)或`Ctrl+J`(Windows)打开AI助手

## 便签功能

- **快速创建**: 按`Cmd+Shift+N`(macOS)或`Ctrl+Shift+N`(Windows)
- **便签整理**: 可自定义快捷键整理所有便签窗口
- **边缘吸附**: 将便签拖到屏幕边缘可自动隐藏

## 知识库

1. 打开"知识库"面板
2. 点击"上传文档"添加参考资料
3. 在写作时，系统会自动检索相关内容
